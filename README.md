# Nicolet FEAR Dark Theme

A dark theme for Visual Studio Code.

## Installation

1. Go to the [Releases](../../releases) page of this repository
2. Download the latest `nicolet-fear-dark-1.0.0.vsix` file
   - Ignore the "Source code" files — those are not needed
4. Open VS Code
5. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
6. Type **Extensions: Install from VSIX…** and select it
7. Navigate to your downloaded `.vsix` file and select it
8. Click **Install**

## Activation

1. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
2. Type **Preferences: Color Theme** and select it
3. Find and select **Nicolet FEAR** from the list

Alternative(Easier):
1. Press **Ctrl-K-Ctrl-T** and select **Nicolet FEAR** from the list
